# Snake Game — 設計理念

經典貪食蛇規則簡單，但我們相信體驗還能更豐富。遊戲保留了「吃食物、變長、別撞牆」核心，在操作與模式上做了大量調整。

方向鍵輸入採用佇列機制，快速連打不會被吞掉或誤判自撞。畫面透過 requestAnimationFrame 插值，蛇身移動流暢而非一格一格跳。

遊戲內建五種模式：Classic（傳統規則）、Free（隨分數加速並生成食物與障礙物）、Speed Survival（極速挑戰附排行）、CTF（雙人搶旗）、Magnet（食物自動物體吸引，吃一生成二）。每種改寫 snake 的某條規則，試探同一核心能延伸出多少玩法。

牆壁碰撞有 40ms 寬限期，避免貼牆瞬間誤判死亡。修改器（Reverse、Portal、Wrap）讓玩家自由組合規則。

這是對經典機制的重新審視——不為了複雜而複雜，而是問「如果 snake 這樣改，會不會更好玩？」
